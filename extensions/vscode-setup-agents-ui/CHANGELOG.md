# Changelog

## [0.1.0] — 2026-04-26

### Added

- Role-based profile selector with 11 Salesforce profiles.
- Multi-tool rule generation: Cursor, GitHub Copilot, Windsurf, Claude Code.
- Scope selector: project-level vs global rules.
- MCP integration configuration: Figma, Jira, Draw.io, GitHub.
- Salesforce org connection from the sidebar.
- SF CLI health check with error banner and retry.
- Rule management: view, delete, and import from URL or file.
- Demo recording pipeline (Playwright + ffmpeg GIF generation).
- 27 E2E tests and 225 unit tests.
